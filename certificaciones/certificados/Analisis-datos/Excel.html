Excel Business Intelligence 1: Obtener y Transformar (Power Query) (365/2019)

"Obtener y Transformar" es una característica poderosa en Excel que te permite importar, transformar y combinar datos de diversas fuentes en un solo lugar. En las versiones 365 y 2019, esta característica se conoce como "Power Query". Power Query facilita la preparación y limpieza de datos antes de analizarlos o visualizarlos en Excel. A continuación, se describen las características clave y cómo funciona "Obtener y Transformar":

Características Clave: Conexión a Fuentes de Datos: Power Query permite conectarse a una variedad de fuentes de datos, incluidas bases de datos, archivos de texto, archivos Excel, servicios en la nube y más.

Importación de Datos: Puedes importar datos desde una o varias fuentes de datos en una sola hoja de Excel o en un nuevo libro.

Transformación de Datos: Puedes realizar transformaciones en los datos para limpiarlos, darles formato y estructura adecuada. Esto incluye eliminar filas vacías, renombrar columnas, filtrar y más.

Operaciones de Combinación: Power Query permite combinar datos de múltiples fuentes utilizando operaciones como uniones (joins), apilamiento y combinación de consultas.

Editor de Consultas: Cada vez que realizas una transformación en los datos, Power Query registra las acciones en el Editor de Consultas, donde puedes ver y editar cada paso.

Lenguaje M: Power Query utiliza el lenguaje de fórmula M para realizar transformaciones de datos. Aunque no es necesario aprender M para usar Power Query, puedes personalizar transformaciones y crear pasos personalizados si lo deseas.

Carga de Datos: Después de transformar los datos según tus necesidades, puedes cargarlos en una hoja de Excel o en un modelo de datos.

Actualización de Datos: Puedes configurar las consultas para que se actualicen automáticamente cuando cambien los datos de origen. Esto es útil para mantener tus informes y análisis actualizados.

Reutilización de Consultas: Una vez que hayas creado una consulta, puedes reutilizarla en otros informes y análisis.

Cómo Usar Power Query (Obtener y Transformar): Abrir Power Query: En Excel, ve a la pestaña "Datos" y selecciona "Obtener Datos" > "De Otras Fuentes" > "Power Query". Esto abrirá el Editor de Consultas.

Conectar a la Fuente de Datos: Selecciona la fuente de datos desde la que deseas importar datos. Esto puede ser una base de datos, un archivo CSV, un archivo Excel, etc.

Aplicar Transformaciones: Utiliza las herramientas en el Editor de Consultas para realizar transformaciones en los datos. Puedes filtrar, ordenar, combinar consultas, agregar columnas personalizadas y más.

Cargar Datos: Después de aplicar las transformaciones, puedes cargar los datos transformados en una hoja de Excel o en un modelo de datos.

Actualización Automática: Si deseas que los datos se actualicen automáticamente, configura la opción de actualización automática en la consulta.

La característica "Obtener y Transformar" (Power Query) en Excel Business Intelligence es una herramienta poderosa para preparar y combinar datos de manera eficiente. Facilita la limpieza y transformación de datos, lo que es esencial para garantizar que los datos utilizados en análisis y reportes sean precisos y útiles.
